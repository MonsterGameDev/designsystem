Below is the Kirby color palette. You can try out the different color combinations there and see how it can be used in the CSS in the gray code example box. All the colors are fetched from the scss/themes/_common-themes.scss file.

Note: NativeScript and Web coloring is set differently, because NativeScript doesn't support CSS variables.

Web example:
.color-sample-example {{'{'}}
  color: var(--{{selectedOnColor}});
  background-color: var(--{{selectedColor}});
{{'}'}}
Native example:
.color-sample-example {{'{'}}
  color: get-color('{{selectedOnColor}}');
  background-color: get-color('{{selectedColor}}');
{{'}'}}
